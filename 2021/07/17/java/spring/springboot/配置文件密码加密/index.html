SpringBoot配置文件密码加密

发表于 2021-07-17 更新于 2021-07-23 分类于 springboot 阅读次数：

前言

在日常开发中，可能很随意把数据库密码直接明文暴露在配置文件中，在开发环境可以这么做，但是在生产环境，是相当不建议这么做，毕竟安全无小事，谁也不知道哪天密码就莫名其妙泄露了。springboot项目中如何对数据库密码进行加密
